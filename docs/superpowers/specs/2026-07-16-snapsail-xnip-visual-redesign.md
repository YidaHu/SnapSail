# SnapSail Xnip 风格视觉重设计

- 日期：2026-07-16
- 状态：用户已确认直接实施
- 视觉方向：Xnip Classic Refined

## 目标

将当前以默认 AppKit 控件为主的功能界面重做为接近 Xnip 的轻量、克制、精致的 macOS 截图体验。重点不是换颜色，而是重建截图选择流程、控件层级、间距、反馈与动画，让截图操作更顺滑、更容易理解。

## 统一视觉语言

- 使用系统浅灰窗口背景、白色内容面和 macOS 蓝色强调色。
- 窗口圆角、阴影和材质遵循系统风格，不使用夸张渐变。
- 文字层级限定为标题、分组标签、正文和辅助说明四级。
- 所有工具栏使用 SF Symbols 图标，文字只作为工具提示或必要标签。
- 控件间距采用 8、12、16、24、32 点节奏。
- 浮层和工具条使用半透明材质、10 至 12 点圆角和柔和阴影。
- 不复制 Xnip 的图标、名称或其他品牌资产。

## 截图选择流程

### 启动

快捷键启动后，所有屏幕覆盖轻度暗化遮罩。遮罩不压黑屏幕，只降低背景干扰。鼠标移动时自动识别窗口，以蓝色细边框和轻微蓝色透明填充突出当前窗口。

### 区域选择

- 鼠标拖动时实时显示清晰选区。
- 左上角显示像素尺寸胶囊标签。
- 鼠标附近显示像素放大镜，包含中心十字线。
- 松开鼠标后不立即截图，选区进入可编辑状态。
- 选区提供四角和四边共八个调整手柄。
- 拖动选区内部可整体移动。
- 方向键每次移动 1 点，Shift 加方向键每次移动 10 点。
- Space 在窗口吸附与自由区域之间切换。
- Esc 取消，Return 确认普通截图。

### 悬浮工具条

选区完成后，在选区下方显示悬浮工具条。空间不足时自动放到选区上方。工具条包含：

1. 完成截图。
2. 滚动截图。
3. 复制到剪贴板。
4. 贴图置顶。
5. 取消。

工具条按钮为图标按钮，悬停显示说明。主要操作使用蓝色选中背景，危险或取消操作保持中性。

### 窗口与多窗口

- 单击窗口直接进入已选择状态并显示工具条，不立即完成。
- Shift 单击增加或移除多个窗口。
- 已选择窗口使用绿色细边框，当前悬停窗口使用蓝色边框。
- Return 完成，Esc 取消。

## 滚动截图体验

- 选区保持蓝色边框，不再显示全屏暗遮罩。
- 选区旁显示窄型实时预览卡片。
- 卡片显示缩略长图、已捕获高度、匹配状态、完成和取消按钮。
- 正常拼接显示绿色状态；没有移动显示灰色等待；匹配失败显示橙色暂停。
- 预览最多每 500 毫秒刷新一次，捕获和匹配在后台串行队列执行。
- 匹配算法采用粗搜加精搜，限制采样区域，避免逐像素穷举造成卡顿。
- 已有长图不在每次采帧时完整重建，仅在预览节流或用户完成时合成。

## 标注编辑器

- 编辑器以图像为视觉中心，背景使用柔和深灰画布。
- 顶部工具栏改为统一图标按钮，不再使用宽度不一的文字按钮。
- 当前工具使用蓝色圆角底色，悬停显示工具名称和快捷键。
- 颜色、线宽、撤销和重做放在同一工具栏右侧。
- Copy、Pin、Save 作为右下角主要操作，Save 使用蓝色主按钮。
- 画布四周保持足够留白，长图自动适配宽度并允许流畅滚动。

## 偏好设置

- 使用接近 Xnip 参考图的顶部图标分页，而不是长单页表单。
- 分页包括 General、Capture、Scrolling、Export 和 Shortcuts。
- 每页使用居中的标签列和控件列，标签右对齐。
- 相关复选框与辅助说明成组排列，减少视觉噪声。
- 保存目录、文件名和快捷键输入使用统一圆角输入框。
- 设置窗口固定合理宽度，不因内容变化而跳动。

## 菜单栏

- 菜单项增加图标、快捷键和清晰分组。
- 截图操作位于首组，历史和设置位于第二组，关于与退出位于末组。
- 菜单栏图标保持单色模板效果。

## 性能标准

- 区域拖动、移动和调整过程中不执行截图或重计算窗口列表。
- 窗口列表在启动选择时获取一次，必要时低频刷新。
- 鼠标移动只更新必要图层，不重绘整个窗口层级。
- 滚动帧处理中最多保留一个待处理帧，落后时丢弃旧待处理帧。
- 实时预览节流为每 500 毫秒一次。
- 普通区域截图从确认到编辑器出现的目标延迟低于 300 毫秒。

## 错误与反馈

- 没有屏幕录制权限时不进入黑色遮罩，先显示权限说明。
- 滚动匹配失败时保持已捕获内容并显示暂停提示。
- 选区太小时保持选区并提示最小尺寸，不直接退出。
- 多屏切换或屏幕变化时安全取消当前选择。

## 验收标准

1. 截图启动后背景遮罩轻盈，窗口悬停和区域拖动无明显卡顿。
2. 区域松开后可以移动、八方向调节并使用悬浮工具条。
3. 放大镜、尺寸标签、选区边框和手柄视觉一致。
4. 普通截图、滚动截图、复制和贴图都能从选区工具条直接进入。
5. 滚动捕获期间主线程保持响应，预览刷新不拖慢拼接。
6. 设置页布局和视觉层级接近用户提供的 Xnip 参考图。
7. 编辑器工具栏使用统一图标并保持图像为视觉中心。
8. 原有截图、标注、历史、保存和快捷键能力不回退。
